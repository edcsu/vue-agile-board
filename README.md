# vue-agile-board

## Project setup

```javascript
// latest stable version of vue if itsnot installed yet
npm install vue
npm install
```

### Compiles and hot-reloads for development

```javascript
npm run serve
```

### Compiles and minifies for production

```javascript
npm run build
```

### Run your tests

```javascript
npm run test
```

### Lints and fixes files

```javascript
npm run lint
```
